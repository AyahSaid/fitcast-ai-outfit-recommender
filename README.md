# FitCast – AI-Powered Weather-Based Outfit Recommendation System

## Overview

FitCast is a full-stack AI-powered mobile application that provides personalized outfit recommendations based on real-time weather conditions, user preferences, activities, and personal comfort levels.

The application combines Machine Learning, rule-based decision making, weather analysis, and personalized feedback to generate intelligent clothing recommendations instead of simply displaying weather forecasts.

This project was developed as a Computer Science senior project at the German Jordanian University.

---

## Features

### Intelligent Outfit Recommendations
- Generates complete outfit suggestions based on current weather conditions.
- Adapts recommendations to different activities and occasions.
- Supports casual, formal, university, office, gym, travel, and other contexts.

### Machine Learning Integration
- Uses a machine learning model to estimate thermal clothing requirements.
- Predicts appropriate clothing layers based on environmental conditions.
- Personalized recommendations improve through user feedback.

### Weather Analysis
- Retrieves real-time weather information.
- Considers:
  - Temperature
  - Humidity
  - Wind Speed
  - Rain Conditions
  - UV Index

### Personalization
- Supports different comfort preferences.
- Allows users to indicate whether they generally feel hot or cold.
- Stores user preferences using Firebase.

### Modesty & Lifestyle Support
- Supports modest outfit configurations.
- Includes personalized outfit filtering based on user preferences.

### AI Assistant Integration
- Uses OpenAI APIs for intelligent user interaction.
- Processes natural language input and outfit-related requests.

### Dynamic Avatar System
- Displays outfit recommendations using a customizable avatar.
- Visualizes clothing combinations directly inside the application.

### User Authentication
- Secure account creation and login.
- Firebase Authentication integration.

### Cloud Database
- User profiles stored using Firebase Firestore.
- Personalized settings synchronized across sessions.

## System Architecture

FitCast follows a modular full-stack architecture:

1. Flutter Mobile Application
2. FastAPI Backend Service
3. Machine Learning Prediction Engine
4. Firebase Authentication
5. Cloud Firestore Database
6. Weather API Integration
7. OpenAI Integration

The frontend communicates with the backend to obtain intelligent outfit recommendations while Firebase manages authentication and user data.
